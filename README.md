# Cortex Project Manager 🧠

Sistema de gestión de proyectos y optimización de carga cognitiva desarrollado con React y Capacitor. Este sistema centraliza flujos de trabajo de ingeniería, robótica y desarrollo personal.

## 🚀 Tecnologías
* Frontend: React + Vite.
* Estilos: Tailwind CSS.
* Mobile: Capacitor (Android Nativo).
* Base de Datos: SQLite (Persistencia local).
* Iconos: Lucide React.

## 🛠️ Instalación y Uso

1. Clonar el repositorio:
   git clone https://github.com/ZakiumyI/cerebro-externo

2. Instalar dependencias:
   npm install

3. Ejecutar en desarrollo:
   npm run dev

4. Sincronizar con Android:
   npm run build
   npx cap sync
   npx cap open android

## 📋 Características
* Diagnóstico de Capacidad: Medidor de energía operativa en tiempo real.
* Gestión de Hilos: Definición de arquitecturas de proyecto con niveles de estrés.
* Cronograma Crítico: Visualización dinámica de fechas de entrega.
